# LinkedIn Request Filter

A Chrome extension that auto-accepts LinkedIn connection requests based on keywords and mutual connections — safely, at human-like speed.

## Features
- Preview who will be accepted before touching anything
- Filter by keywords in bio (founder, VC, angel, YC, etc.)
- Filter by minimum mutual connections
- Human-like random delays (2–6s) between clicks
- Session cap of 25 accepts to stay safe

## Install
1. Download the ZIP or clone this repo
2. Go to `chrome://extensions` → enable Developer Mode
3. Click Load unpacked → select the `lrf` folder
4. Go to `linkedin.com/mynetwork/invitation-manager/received/`
5. Click the extension icon → Preview → Accept

## Usage
1. Set your keywords and minimum mutuals
2. Click **Preview matches first** — review who matches
3. Click **Accept** to confirm, or adjust filters and preview again
